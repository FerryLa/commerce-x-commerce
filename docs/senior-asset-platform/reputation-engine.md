# Reputation Engine 설계 — SourceCred 응용

> "시니어의 존엄을 데이터 자산으로 복권시키는 기술"

---

## 1. SourceCred란?

SourceCred는 오픈소스 커뮤니티의 기여도를 자동으로 수치화하는 알고리즘입니다.
원래는 GitHub 커밋/PR/리뷰를 점수화했지만, 이 플랫폼에서는 **삶의 기여 이벤트**에 응용합니다.

---

## 2. 기여 이벤트 유형 및 점수

| 이벤트 | 기본 점수 | 가중치 요소 |
|---|---|---|
| 멘토링 1회 완료 | 10점 | × 만족도(1~5) / 5 |
| 커뮤니티 상담 | 5점 | × 참여자 수 |
| 문제 해결 기여 | 15점 | × 성공률(0~1) |
| 지역 네트워크 연결 | 8점 | 고정 |
| 세대간 프로젝트 참여 | 20점 | × 완성도(0~1) |

### 최종 점수 공식

```
reputation_score = Σ(base_score × weight_factor) × decay_factor(time)
```

- `decay_factor`: 최근 활동일수록 가중치 높음 (반감기 90일)
- 상한: 100점 (정규화)

## 4. 계산 예시

**김○○ 시니어 (65세, 군산)**

| 이벤트 | 날짜 | 기본점수 | 가중치 | 획득점수 |
|---|---|---|---|---|
| 주니어A 멘토링 (3회) | 2025-10 | 10×3 | ×(4.8/5) | 28.8 |
| 창업 불안 문제 해결 | 2025-11 | 15 | ×0.9 | 13.5 |
| 지역 네트워크 연결 | 2026-01 | 8 | ×1.0 | 8.0 |
| **합계 (decay 적용 후)** | | | | **87점** 🏆 |

---

## 5. 투명성 원칙

- 모든 기여 이벤트는 **기록 불변(append-only)** 으로 저장
- 시니어 본인이 자신의 점수 산출 근거 조회 가능
- 주니어가 만족도 평점 후 **상호 동의** 시 반영
- 점수는 일자리 매칭에만 사용 (신용 점수 등 외부 활용 금지)

---

## 6. 향후 확장

- **블록체인 기록**: 기여 이력의 위변조 방지 (Hyperledger Fabric)
- **토큰화**: 점수를 지역 화폐로 전환 가능 (지자체 협력)
- **연금 연계**: 활동 기여 → 노인 일자리 급여 보완
